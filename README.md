Установка и использование:
1. Скопировать проект на сервер, отредактировать в index.php путь к Yii
2. Создать базу данных в MySQL: "create database phonebook character set utf8 collate utf8_general_ci;"
3. Отредактировать database.php, если нужно
4. Запустить миграции: protected/yiic migrate

5. Зайти на сайт, кликнуть на Phonebook
6. Данные администратора: имя - admin, пароль - admin.
