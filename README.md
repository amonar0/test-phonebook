# Установка

* Скопировать проект на сервер, отредактировать в index.php путь к Yii
* Создать базу данных в MySQL: "create database phonebook character set utf8 collate utf8_general_ci;"
* Отредактировать database.php, если нужно
* Запустить миграции: protected/yiic migrate

# Использование

* Зайти на сайт, кликнуть на Phonebook
* Данные администратора: имя - admin, пароль - admin.
